Memoria Ram

In elettronica e informatica la RAM ( Random Access Memory ovvero memoria ad accesso casuale) è un tipo di memoria volatile caratterizzata dal permettere l'accesso diretto a qualunque indirizzo di memoria con lo stesso tempo di accesso.Nella memoria RAM vengono copiati (caricati) i programmi che la CPU deve eseguire. Una volta chiuso il programma, le modifiche effettuate, se non opportunamente salvate sul disco rigido o su altra memoria non volatile, verranno perse.Per le sue caratteristiche, la RAM viene utilizzata come memoria primaria nei computer più comuni.
